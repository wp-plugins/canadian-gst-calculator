Add a free simple customizable Canadian GST calculator to your web site. 

== Description ==

This is simple Canadian [GST calculator](http://gstcalculator.ca) for Goods and Services Tax calculation in Canada.

Calculator is very easy customizable: you can change color of background, borders and text to match your web site's theme and change widget title.

== Installation ==

1. Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance -> Widgets and add the widget to your website sidebar

Please visit [plugin home page](http://gstcalculator.ca/widgets) for more detailed installation and setup instructions
